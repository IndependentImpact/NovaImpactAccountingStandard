# Reference

This directory contains reference assets used to understand, compare, or bridge
external and legacy representations into the NIAS model.

## What is here

- Versioned `*.json` reference schemas for artefacts such as PDD sections,
  reports, and supporting entities.
- `*.R` transformation scripts for converting reference payloads into
  Fluree-oriented or NIAS-aligned structures.
- `*.docx` template documents and supporting reference files.

## When to work here

- Use this directory when you need examples, legacy mappings, or document
  templates.
- Avoid casual edits here unless you are intentionally updating reference or
  compatibility assets.
