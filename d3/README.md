# D3 visualisations

This directory contains browser-viewable D3-based visualisations of NIAS
semantic content.

## What is here

- `IIglossaryd3.html` provides an interactive view of glossary content.
- `IIontologyd3.html` provides an interactive view of ontology content.

## When to work here

- Use these files to explore or demonstrate the model in a browser.
- Treat them as visualisation/support assets rather than the canonical semantic
  source of truth.
