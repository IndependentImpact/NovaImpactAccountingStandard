# Knowledge domains

This directory contains knowledge-domain concept schemes referenced by the Nova
Impact Accounting Standard.

## What is here

- `GHGKnowledgeDomains.ttl` defines greenhouse-gas-related knowledge domains in
  Turtle.

## When to work here

- Edit this directory when the taxonomy of required subject-matter expertise
  changes.
- Treat these files as canonical semantic source material, not generated output.
