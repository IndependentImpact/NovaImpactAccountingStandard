# Contributors

Thank you to everyone who has contributed to the Nova Impact Accounting Standard.

## How to Contribute

Contributions are welcome. Please open an issue or pull request in the [GitHub repository](https://github.com/IndependentImpact/NovaImpactAccountingStandard).

Before contributing, please review:

- `README.md` — project overview and structure.
- `dev/InProcess/` — active development plans and specifications.
- `dev/Completed/` — completed design decisions and archived plans.

## Contributors

See the [GitHub contributors page](https://github.com/IndependentImpact/NovaImpactAccountingStandard/graphs/contributors) for a full list of contributors.
